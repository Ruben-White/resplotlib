resplotlib
==========

Resplotlib is a library for resilient plotting of geospatial data in Python. It is a wrapper around the plotting functionalities of `xarray <http://xarray.pydata.org/en/stable/>`_, `xugrid <https://deltares.github.io/xugrid/>`_, `geopandas <https://geopandas.org/>`_, `contextily <https://contextily.readthedocs.io/en/latest/>`_, `matplotlib <https://matplotlib.org/stable/index.html>`_, and `ipyleaflet <https://ipyleaflet.readthedocs.io/en/latest/>`_.

GitHub: https://github.com/Ruben-White/resplotlib
