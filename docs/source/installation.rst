Installation
============

1. Use `uv <https://docs.astral.sh/uv/>`__ to install the package from `PyPI <https://pypi.org/project/resplotlib/>`__:

   .. code-block:: bash

      uv add resplotlib  

2. Use `pip` to install the package from `PyPI <https://pypi.org/project/resplotlib/>`__:

   .. code-block:: bash

      pip install resplotlib

3. Use `conda` to install the package from `conda-forge <https://anaconda.org/conda-forge/resplotlib>`__:

   .. code-block:: bash

      conda install -c conda-forge resplotlib
